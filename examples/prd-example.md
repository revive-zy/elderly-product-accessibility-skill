# PRD Example: 中老年学习 App

## 用户分层

- 轻度适老用户: 会使用微信和短视频，希望低压力学习。
- 中度适老用户: 会点按钮和听语音，但对搜索、分类、设置不熟。
- 协助者用户: 子女或社区人员，帮助安装、配置和解释。

## 核心任务

1. 打开 App 后直接开始今天的学习。
2. 听清楚词语读音。
3. 看懂生活化例句。
4. 确认自己认识了。
5. 学完后可选继续，不形成压力。

## 首页结构

- 顶部: 今日学习状态。
- 主区: 今日 3 个新词。
- 主动作: `开始今天学习`。
- 辅助入口: `语音播放`、`字体大小`、`联系客服/家人协助`。

## 核心流程

```text
打开 App -> 生活照片 -> 大字词 -> 朗读 -> 例句 -> 用户确认 -> 轻量复习 -> 完成页
```

## 验收标准

- 首页只有一个主动作。
- 核心按钮点击区域不小于 60 x 60 dp/pt。
- 正文不小于 18 dp/pt。
- 学完后展示正反馈，继续学习是可选项。
- 不出现排名、考试、失败惩罚和诱导付费。
