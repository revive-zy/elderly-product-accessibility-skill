# UI_SPEC Example: 适老化首页

## 信息层级

1. 今日状态: 用户现在是否有待完成任务。
2. 主任务入口: 一个明确按钮。
3. 辅助入口: 语音、客服、家人协助、字体设置。

## 字体

- 页面标题: 24 到 28 sp/pt。
- 正文: 18 到 22 sp/pt。
- 按钮文字: 20 到 24 sp/pt。
- 支持系统字体 1.5x 和 2x 放大，不截断、不重叠。

## 触控区域

- 主按钮高度建议 56 到 64 dp/pt。
- 主要组件点击区域不小于 60 x 60 dp/pt。
- 次要组件点击区域不小于 44 x 44 dp/pt。

## 颜色和对比度

- 普通文本对比度不低于 4.5:1。
- 大字号文本对比度不低于 3:1。
- 错误、成功、警告不能只依赖颜色表达。

## 状态

- Loading: 说明正在处理什么。
- Success: 明确已完成什么。
- Error: 说明失败原因、重试路径和客服入口。
- Empty: 说明为什么为空和下一步可做什么。
